# 🧱 Programación Orientada a Objetos en Java (POO)

Repositorio con ejercicios prácticos enfocados en la **Programación Orientada a Objetos (POO)** en Java, aplicando conceptos como clases, objetos y organización por paquetes.

---

## 📚 Contenido

Este proyecto incluye ejemplos sobre:

* 🧩 Creación de clases
* 🧍 Instanciación de objetos
* 📦 Uso de paquetes
* 🔒 Encapsulamiento básico
* 🔁 Reutilización de código

---

## 🧩 Módulos del proyecto

El proyecto está organizado en diferentes paquetes, cada uno representando un caso práctico:

* 🐾 **animales** → Ejemplo de clases de animales
* ➗ **aritmetica** → Operaciones básicas
* 📚 **biblioteca** → Sistema de libros
* 🎓 **estudiantes** → Gestión de estudiantes
* 🐶 **mascotas** → Manejo de objetos mascota
* 👤 **persona** → Datos de persona
* 🛒 **ventas** → Sistema de ventas
* 🎮 **videojuego** → Ejemplo de objetos en juegos

---

## ▶️ Cómo ejecutar

1. Clonar el repositorio:

```bash
git clone https://github.com/tu-usuario/ClasesObjetos-POO-Java.git
```

2. Abrir en tu IDE (VS Code / IntelliJ)

3. Ejecutar las clases de prueba dentro de cada paquete
